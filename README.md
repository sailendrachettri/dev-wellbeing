# Dev Wellbeing

**Dev Wellbeing** is a digital wellbeing application designed specifically for developers to help them understand, track, and improve their daily work habits. It automatically monitors application usage, coding time, and activity patterns, presenting the data through clean and intuitive visualizations such as daily and weekly timelines.
